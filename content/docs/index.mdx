Hoshimi is a TypeScript-first Lavalink v4 client focused on clean architecture, strict typing and practical extension points.

## Why Hoshimi?

- Typed API surface exported from src/index.ts.
- Clear separation between manager, node, player, queue and storage.
- Ready for custom structures and adapter-based persistence.

## Install

```bash
pnpm add hoshimi
```

## Quick Setup

```ts twoslash copy wrap title="src/index.ts"
const gatewaySend = async (_guildId: string, _payload: unknown): Promise<void> => {};
// ---cut---
import { Hoshimi } from 'hoshimi';

const hoshimi = new Hoshimi({
  nodes: [
    {
      host: 'localhost',
      port: 2333,
      password: 'youshallnotpass',
    },
  ],
  sendPayload(guildId, payload) {
    // Forward payload to your gateway implementation.
    return gatewaySend(guildId, payload);
  },
});
```

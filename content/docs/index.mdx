Hoshimi is a TypeScript-first Lavalink v4 client built with strict typing, clean architecture, and extensibility in mind.

## Why Hoshimi?

- **Type-safe**: Comprehensive typings and type guards prevent runtime errors.
- **Modular**: Separate concerns for nodes, players, queues, and storage—override safely.
- **Scalable**: Built for bots handling multiple nodes, resumable sessions, and persistence.

## Install

```bash
pnpm add hoshimi
```

## First Steps

```ts twoslash copy wrap title="src/index.ts"
const gatewaySend = async (guildId: string, payload: unknown): Promise<void> => {  };
// ---cut---
import { Hoshimi } from 'hoshimi';

const hoshimi = new Hoshimi({
  nodes: [
    {
      host: 'localhost',
      port: 2333,
      password: 'youshallnotpass',
    },
  ],
  sendPayload(guildId, payload) {
    return gatewaySend(guildId, payload);
  },
});
```

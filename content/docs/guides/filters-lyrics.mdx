## Audio Filters

Apply built-in filters or reset them. Common presets include nightcore, karaoke, and more.

```ts twoslash copy wrap title="audio-filters.ts"
import type { Player } from 'hoshimi';

declare const player: Player;

await player.filterManager.setNightcore({
  rate: 1.25,
  pitch: 1.25,
  speed: 1.25,
});

await player.filterManager.setKaraoke({
  level: 1.0,
  monoLevel: 1.0,
  filterBand: 220,
  filterWidth: 100,
});

// Reset all filters
await player.filterManager.reset();
```

## Lyrics Subscription

Subscribe to lyrics for the current track. Handle errors gracefully.

```ts twoslash copy wrap title="lyrics-subscription.ts"
import type { Player } from 'hoshimi';

declare const player: Player;

const lyrics = await player.lyrics.current();

if (lyrics && !('error' in lyrics)) {
  await player.data.set('lyrics', lyrics);
  await player.data.set('enabledLyrics', true);
  await player.lyrics.subscribe();
}
```

## Cleanup on Session End

Remove subscriptions and clear stored state when playback stops or the player is destroyed.

```ts twoslash copy wrap title="lyrics-cleanup.ts"
import type { Player } from 'hoshimi';

declare const player: Player;

const lyricsId = await player.data.get('lyricsId');
if (lyricsId) {
  const enabled = !!(await player.data.get('enabledLyrics'));

  if (enabled) {
    await player.lyrics.unsubscribe();
  }

  await player.data.delete('lyricsId');
  await player.data.delete('lyrics');
  await player.data.delete('enabledLyrics');
}
```

## Important Notes

- **Filter managers**: `plugin` is the Lavalink plugin filter manager; `dspx` is the DSPX plugin filter manager.
- **Filter conflicts**: Always reset filters when switching presets aggressively.
- **Lyrics errors**: Treat provider errors as expected runtime cases and fallback gracefully.
- **Cleanup timing**: Clear lyrics on `PlayerDestroy` and `QueueEnd` to prevent stale subscriptions.

## Search & Queue

Search for tracks and add them to the queue.

```ts twoslash copy wrap title="search-and-queue.ts"
import { LoadType } from 'hoshimi';
import type { Player } from 'hoshimi';

declare const player: Player;

const result = await player.search({
  query: 'Daft Punk - Harder Better Faster Stronger',
  requester: {
    id: '123',
    username: 'demo-user',
  },
});

if (result.loadType === LoadType.Track || result.loadType === LoadType.Search) {
  await player.queue.add(result.tracks[0]);
}
```

## Playback Control

Play, pause, skip, seek, adjust volume, and set loop modes.

```ts twoslash copy wrap title="playback-controls.ts"
import { LoopMode } from 'hoshimi';
import type { Player } from 'hoshimi';

declare const player: Player;

// Connect and play
if (!player.connected) await player.connect();
if (!player.isPlaying()) await player.play();

// Control playback
await player.setPaused(true);    // pause
await player.setPaused(false);   // resume
await player.seek(30_000);       // seek to 30s
await player.skip({ to: 1 });    // skip to next
await player.setVolume(150);     // 150% volume

// Loop modes
player.setLoop(LoopMode.Queue);  // loop entire queue
player.setLoop(LoopMode.Track);  // loop current track
player.setLoop(LoopMode.Off);   // disable looping
```

## Disconnect vs Destroy

Disconnection stops playback but keeps the player. Destruction removes the player entirely.

```ts twoslash copy wrap title="session-teardown.ts"
import { DestroyReasons } from 'hoshimi';
import type { Player } from 'hoshimi';

declare const player: Player;

// Disconnect from voice (stops playback, keeps player)
await player.disconnect();

// Destroy player (cleanup, removal)
await player.destroy(DestroyReasons.Stop);
```

## Important Notes

- **`player.connected`**: Voice connection state, not playback state.
- **`player.playing` vs `player.paused`**: Different signals; use both to understand player state.
- **`player.position`**: Computed value; use `seek()` to move the playback position.
- **`player.isPlaying()`**: Check before calling `play()` to avoid redundant operations.
